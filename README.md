# Weather-Website
Explore real-time weather updates with our HTML, CSS, and JavaScript-powered website. 

## Features-
- Display current weather conditions.
- Show temperature, humidity, and wind speed.
- Search for weather information by city.

## Technologies Used-
- HTML
- CSS
- JavaScript
